# Minions Qa-reports

Aggregated test reports, health dashboards, trend analysis, and regression alerts

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-qa-reports/sdk`) — Core library for Node.js
- **Python SDK** (`minions-qa-reports`) — Python bindings
- **CLI Tool** (`@minions-qa-reports/cli`) — Command-line interface
- **Web App** — Interactive playground at [qa-reports.minions.wtf](https://qa-reports.minions.wtf)
