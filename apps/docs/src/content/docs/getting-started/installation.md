---
title: Installation
description: Install Minions Qa-reports packages
---

## TypeScript / Node.js

```bash
npm install @minions-qa-reports/sdk minions-sdk
```

## Python

```bash
pip install minions-qa-reports
```

## CLI

```bash
npm install -g @minions-qa-reports/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
